# pocketpy-unity-plugin

> Due to updated regional licensing, distribution, and compliance policy requirements for publishers in the Greater China Region (including Hong Kong and Macau), one or more of your Asset Store assets will no longer be available for purchase on the Global Unity Asset store, effective March 31, 2026.

According to the notification from Unity, this asset will be unavailable for purchase on the Global Unity Asset store. So I decide to open source it. Anyone can use it for free under the GPLv3 licence. Purchased users can still use it under the license they bought.

See [Documentations](PocketPython/Documentations/introduction.md) for more details about how to use this plugin.

If you want to contribute to this project, please feel free to fork and submit pull requests. I will review and merge them if they are good.
